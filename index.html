Introduction

<<Hack|Track GNU/Linux is an open source operating system developed by the HTGL Project from Indonesia that provides penetration testing.

It includes a complete arsenal of portable weapons for IT security and digital forensic operations. It also includes everything you need to develop your own program or protect your privacy while surfing the internet.

HackTrack contains industry-specific modifications as well as several hundred tools targeted for various Information Security tasks, such as Penetration Testing, Security Research, Computer Forensics, Reverse Engineering, Vulnerability Management and Red Team Testing. HackTrack is a multi-platform, accessible and freely available solution for information security professionals and enthusiasts.

HackTrack Features

  • More than 600 penetration testing tools ready to use.
  • It's free and always will be hackTrack is completely free and always will be. You will never have to pay for HackTrack.
  • Open source Git tree We are committed to an open source development model and our development tree is available for all to see. All source code that goes into HackTrack is available to anyone who wants to modify or rebuild the package to suit their specific needs.
  • Extensive wireless device support A regular sticking point with Linux distributions is support for wireless interfaces. We've built HackTrack to support as many wireless devices as possible, allowing it to run well on a wide range of hardware and making it compatible with many USB and other wireless devices.
  • Custom kernel, patched for injection. As penetration testers, the development team often needs to do wireless assessments, so our kernel includes the latest injection patches.
  • Developed in a safe environment. The HackTrack team is made up of a small group of individuals who are the only ones trusted to package and interact with the repository, all of which is done using multiple secure protocols.
  • GPG-signed packages and repositories. Every package on HackTrack is signed by every individual developer who built and committed it, and the repository then signs the package as well.
  • Multi-language support: While penetration tools tend to be written in English, we've made sure that HackTrack includes true multilingual support, allowing more users to operate in their native language and find the tools they need for the job.
  • Fully customizable. We totally understand that not everyone will agree with our design decisions, so we've made it as easy as possible for our more adventurous users to customize HackTrack to their liking, right down to the kernel.
